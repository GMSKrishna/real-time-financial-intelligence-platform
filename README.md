# 🚀 Real-Time Financial Intelligence Platform

A full-stack real-time financial analytics platform built using FastAPI, PostgreSQL, ETL pipelines, Streamlit, and automated scheduling.

---

# 📌 Features

✅ Real-time crypto market data ingestion  
✅ PostgreSQL database integration  
✅ ETL data processing pipeline  
✅ FastAPI backend APIs  
✅ Swagger API documentation  
✅ Interactive Streamlit dashboard  
✅ Automated scheduler pipeline  
✅ Logging system  
✅ Real-time analytics visualization

---

# 🛠️ Tech Stack

- Python
- FastAPI
- PostgreSQL
- Pandas
- SQLAlchemy
- Streamlit
- Schedule
- Uvicorn
